# Итоговый отчёт по ЛР-02: Транспортная задача

## Кейс 1: Лекарства в районные больницы (`civil_01`)

### 1. Таблица исходных данных и проверка баланса

| Склад | Запас |

| Склад A | 30 |
| Склад B | 40 |

| **Итого запас:** 105 |  |

| Больница | Спрос |
| --- | --- |

| Больница 2 | 25 |
| Больница 3 | 30 |
| Больница 4 | 30 |
| **Итого спрос:** 105 |  |

**Проверка баланса:** Сумма запасов (105) равна сумме спроса (105). Задача сбалансирована (закрытая). Фиктивные узлы не требуются.

### 2. Математическая постановка

Минимизируем суммарные транспортные расходы:


$$\min Z = \sum_{i=1}^{3} \sum_{j=1}^{4} c_{ij} x_{ij}$$


При ограничениях:

* По поставщикам: $\sum_{j=1}^{4} x_{ij} = S_i$ (для $i=1, 2, 3$)
* По потребителям: $\sum_{i=1}^{3} x_{ij} = D_j$ (для $j=1, 2, 3, 4$)
* Неотрицательность: $x_{ij} \ge 0$

### 3. Реализация (код)

Для решения использовался метод `highs` библиотеки `scipy.optimize.linprog`.

```python
import numpy as np
from scipy.optimize import linprog

costs = np.array([[5, 7, 6, 10], [8, 4, 5, 7], [6, 6, 4, 5]])
supplies = np.array([30, 40, 35])
demands = np.array([20, 25, 30, 30])

c = costs.flatten()
num_supp, num_cons = costs.shape
A_eq = []
# Строки (запасы)
for i in range(num_supp):
    row = np.zeros((num_supp, num_cons)); row[i, :] = 1; A_eq.append(row.flatten())
# Столбцы (спрос)
for j in range(num_cons):
    col = np.zeros((num_supp, num_cons)); col[:, j] = 1; A_eq.append(col.flatten())

b_eq = np.concatenate([supplies, demands])
res = linprog(c, A_eq=A_eq, b_eq=b_eq, bounds=(0, None), method="highs")
plan = res.x.reshape(num_supp, num_cons)

```

| --- | --- | --- | --- | --- |
| **Склад A** | 20 | 0 | 10 | 0 |
| **Склад B** | 0 | 25 | 0 | 15 |
| **Склад C** | 0 | 0 | 20 | 15 |

**Минимальная стоимость:** 520.
**Интерпретация:** Алгоритм выбрал маршруты с наименьшими удельными затратами. Нулевые значения в плане указывают на экономически невыгодные маршруты.
